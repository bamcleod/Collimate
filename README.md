# Collimate

Software for collimating wide-field telescopes.

## Getting Started

### Dependencies

* This software currently runs under Linux only as it contains a pre-compiled program.
* It requires having the following python packages installed:
  * numpy
  * matplotlib
  * pyds9
  * astropy
  * yaml
